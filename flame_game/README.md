# Space Shooter Game

Welcome to the Space Shooter Game, a classic arcade game reimagined with Flutter and the Flame engine. This game showcases the power of Flutter and Flame in creating engaging and interactive games.

## Features

- **Flutter and Flame Integration**: Utilizes Flutter's powerful UI framework and the Flame engine for game development.
- **Animations and Inputs**: Supports gesture, mouse, and keyboard controls for a smooth gameplay experience.
- **Collision Detection**: Includes collision detection for bullets and enemies, enhancing the game's realism.
- **Game Components**: Uses Flame's component system to manage game elements like the player, bullets, and enemies.

## Getting Started

To get started with the Space Shooter Game, follow these steps:

1. **Install Flutter**: Ensure you have Flutter installed on your system. If not, follow the official Flutter installation guide.

2. **Install Dependencies**: Navigate to the project directory and run `flutter pub get` to install the necessary Flutter and Flame dependencies.

3. **Run the Game**: Execute `flutter run` to start the game.

## Gameplay

- **Player Movement**: Use gestures or keyboard inputs to move your spaceship.
- **Shooting**: Tap or press the spacebar to shoot bullets at incoming enemies.
- **Enemies and Bullets**: Dodge enemies and try to destroy them with your bullets.
